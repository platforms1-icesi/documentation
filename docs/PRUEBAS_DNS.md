| Entorno / Dispositivo                                   | Objetivo de la Prueba                                                                       | Comando Completo                                                              |
| :------------------------------------------------------ | :------------------------------------------------------------------------------------------ | :---------------------------------------------------------------------------- |
| **Servidor DNS (Linux)**<br>*(Desde SSH `dns-primary`)* | **1. Verificar Autoridad (SOA)**<br>Confirma que el servidor es el dueño de la zona.        | `dig @192.168.20.10 plataformas.local SOA`                                    |
|                                                         | **2. Verificar Delegación (NS)**<br>Confirma redundancia (Primario + Secundario).           | `dig @192.168.20.10 plataformas.local NS`                                     |
|                                                         | **3. Resolución de Servicios**<br>Prueba que las IPs de servicios responden.                | `dig @192.168.20.10 www.plataformas.local`                                    |
|                                                         | **4. Verificar Llaves Públicas**<br>Confirma que las llaves ZSK/KSK existen.                | `dig @192.168.20.10 plataformas.local DNSKEY +short`                          |
|                                                         | **5. Validación DNSSEC**<br>Verifica firma criptográfica en respuestas.                     | `dig @192.168.20.10 plataformas.local +dnssec`                                |
| **Cliente Windows**<br>*(CMD / Símbolo del Sistema)*    | **6. Resolución Básica**<br>Prueba conectividad simple desde cliente.                       | `nslookup www.plataformas.local`                                              |
|                                                         | **7. Consultar SOA desde Windows**<br>Ver quién manda en la zona.                           | `nslookup -type=SOA plataformas.local`                                        |
|                                                         | **8. Forzar Consulta a Secundario**<br>Probar que el esclavo responde si el primario falla. | `nslookup www.plataformas.local 192.168.20.11`                                |
| **Cliente Windows**<br>*(PowerShell)*                   | **9. Validar Autenticidad (DNSSEC OK)**<br>Prueba avanzada de seguridad.                    | `Resolve-DnsName -Name www.plataformas.local -Server 192.168.20.10 -DnssecOk` |
|                                                         | **10. Ver Llaves desde Cliente**<br>Verifica visibilidad pública de llaves.                 | `Resolve-DnsName -Name plataformas.local -Type DNSKEY -Server 192.168.20.10`  |
