| Punto de Montaje | Sistema de Archivos | Tamaño Sugerido | Descripción                                                                                                                                              |
|------------------|---------------------|-----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| /boot            | ext4                | 500 MB          | Archivos de arranque del kernel. Se separa para evitar corrupción.                                                                                                       |
| / (Raíz)         | ext4                | 10 GB           | Contiene el sistema operativo y las aplicaciones (Postfix, Dovecot). Se mantiene limpio de datos de usuario.                                                             |
| swap             | swap                | 4 GB            | Memoria de intercambio. Vital si el antivirus de correo (ClamAV) consume mucha RAM de golpe.                                                                             |
| /var             | ext4                | 30 GB           | El más importante. Aquí viven /var/mail (correos) y /var/spool (cola de envío). Se le da el mayor espacio y se aísla para que si se llena, no tumbe el sistema.         |
| /var/log         | ext4                | 5 GB            | Logs de auditoría (/var/log/mail.log). Separado para evitar ataques de DoS que intenten llenar el disco con registros de error.                                         |
| Espacio Libre    | -                   | (Restante)      | Se deja espacio sin asignar en el Grupo de Volúmenes (VG) para emergencias.                                                                                              |
